# COVID-19 Dashboard Interactivo

Este proyecto muestra un dashboard interactivo de COVID-19 con:

- Casos diarios y acumulados
- Fallecimientos y recuperaciones
- Gráficas interactivas con Plotly
- Predicción simple de casos diarios con regresión lineal
- Selección de país y rango de fechas

## Cómo ejecutar

1. Instalar dependencias:
pip install -r requirements.txt

2. Ejecutar el dashboard:
python covid_dashboard.py
